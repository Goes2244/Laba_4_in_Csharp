# Задание 7. Перегрузка операций

## 7. Название класса: Time; Методы: Унарные операции: ++ добавление минуты к объекту типа Time. -- вычитание минуты из объекта типа Time Операции приведения типа: int (неявная) – результатом является количество минут (время переводится в минуты); bool (явная) – результатом является true, если часы и минуты не равны нулю, и falseв противном случае. Бинарные операции: < Timet1, Timet2 — время переводится в минуты, результатом является true, если количество минут в левом операнде меньше, чем количество минут в правом операнде, и false–в противном случае. > Timet1, Timet2 — время переводится в минуты, результатом является true, если количество минут в левом операнде больше, чем количество минут в правом операнде, и false–в противном случае.
Решение:

Унарные операторы:

  ++ : добавить 1 минуту
  
  -- : вычесть 1 минуту

Бинарные операторы:

  + : сложение времени с минутами/временем
  - : вычитание минут/времени
  < > : сравнение через преобразование в минуты

Операторы приведения:

  bool : true если время не 00:00
  
  int : общее количество минут
